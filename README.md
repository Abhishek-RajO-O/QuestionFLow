# QuestionFlow

**QuestionFlow** is a clean and user-friendly **Question and Answer platform** built to help users ask questions, share answers, and learn from community discussions. The platform provides a simple and intuitive interface where users can easily explore questions, contribute answers, and engage with shared knowledge.

This project focuses on creating a smooth user experience with a structured layout that makes questions and answers easy to read and navigate.

---

## 🚀 Features

* Ask questions easily
* Post answers to help others
* Clean and responsive user interface
* Organized question and answer display
* Easy navigation between discussions
* Community-driven knowledge sharing

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/Abhishek-RajO-O/QuestionFLow.git
```

2. Navigate to the project folder

```
cd QuestionFLow
```

3. Install dependencies

```
npm install
```

4. Run the development server

```
npm start
```

5. Open in browser

```
http://localhost:3000
```
